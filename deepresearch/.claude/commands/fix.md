# /fix - 修复问题

根据扫描发现的问题进行修复。

## 使用方式

```
/fix              # 交互式选择要修复的问题
/fix --all        # 修复所有可自动修复的问题
/fix #3           # 修复第 3 个问题
```

## 执行流程

1. 读取 `data/findings.json` 中的问题
2. **Coder Agent** 分析问题并生成修复方案
3. **Reviewer Agent** 验证修复是否正确
4. 应用修复并更新 findings

## 注意事项

- 修复前会显示变更预览
- 高风险修复需要手动确认
- 修复后建议重新运行 `/scan` 验证
